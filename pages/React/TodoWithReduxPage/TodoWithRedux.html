    لیست وظایف پیشرفته

    مشاهده فایل پروژه در گیت هاب

    در این نسخه پیشرفته از برنامه Todo List از Redux Toolkit و redux-persist برای مدیریت حرفه‌ای وضعیت و ذخیره‌سازی پایدار داده‌ها استفاده شده است.

    🎯 ویژگی‌های جدید:

    • مدیریت وضعیت مرکزی با Redux Toolkit
    • ذخیره‌سازی وظایف با redux-persist در Local Storage
    • ویرایش و حذف وظایف به‌صورت ماژولار
    • نشانه‌گذاری وظایف انجام شده

    🛠️ تکنولوژی‌های استفاده شده:

    • React: برای ساخت کامپوننت‌های UI
    • Redux Toolkit: برای مدیریت وضعیت
    • redux-persist: برای ذخیره‌سازی داده‌ها در مرورگر
    • TailwindCSS: طراحی سبک و واکنش‌گرا

    🔍 جزئیات ساختاری:

    کامپوننت‌ها به صورت جداگانه طراحی شده‌اند و وضعیت وظایف با استفاده از Redux مدیریت می‌شود. با redux-persist، وظایف حتی پس از رفرش یا بستن مرورگر نیز باقی می‌مانند. این ساختار توسعه‌یافته باعث می‌شود برنامه مقیاس‌پذیر و قابل نگهداری باشد.

    💻 راهنمای استفاده:

    1. برای افزودن وظیفه، متن را نوشته و Enter بزنید.
    2. برای ویرایش، روی آیکون مداد کنار وظیفه کلیک کنید.
    3. برای حذف، روی آیکون ضربدر کلیک کنید.
    4. برای علامت‌گذاری به عنوان انجام شده، روی تیک کلیک کنید.
