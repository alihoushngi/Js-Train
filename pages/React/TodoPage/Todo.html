    لیست وظایف

    مشاهده فایل پروژه در گیت هاب

    این برنامه با استفاده از React و TailwindCSS طراحی شده است تا بتوانید به آسانی وظایف روزمره خود را مدیریت کنید.

    🎯 ویژگی‌ها:

    • افزودن وظایف جدید با یک کلیک
    • امکان ویرایش و حذف وظایف
    • نشانه‌گذاری وظایف انجام شده با تیک
    • ذخیره‌سازی وظایف در Local Storage برای حفظ داده‌ها حتی پس از بستن مرورگر

    🛠️ تکنولوژی‌های استفاده شده:

    • React: برای ساختار و مدیریت کامپوننت‌ها
    • TailwindCSS: برای طراحی ظاهری سریع و پاسخ‌گو
    • Local Storage: برای ذخیره‌سازی داده‌ها در مرورگر

    🔍 جزئیات برنامه:

    این برنامه به گونه‌ای طراحی شده که هر بخش آن یک کامپوننت جداگانه دارد. این ساختار ماژولار به شما امکان می‌دهد تا به راحتی بخش‌های مختلف برنامه را ویرایش و سفارشی‌سازی کنید. با استفاده از Local Storage، اطلاعات مربوط به وظایف شما ذخیره می‌شود و حتی بعد از بستن مرورگر هم از بین نمی‌رود. با این قابلیت، مدیریت وظایف روزمره‌تان ساده‌تر از همیشه خواهد بود.

    💻 راهنمای استفاده:

    1. برای افزودن متن خود را بنویسید و دکمه Enter را فشار دهید.
    2. برای ویرایش وظیفه، روی آیکون ویرایش کنار هر وظیفه کلیک کنید.
    3. برای حذف یک وظیفه، روی آیکون ضربدر کلیک کنید.
    4. برای علامت‌گذاری یک وظیفه به عنوان انجام شده، روی تیک کنار آن کلیک کنید.
